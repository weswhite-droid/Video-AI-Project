## ✅ Step-by-Step Instructions

### 1. Go to GitHub Pages Settings

**Open this URL in your browser:**
```
https://github.com/weswhite-droid/Video-AI-Project/settings/pages
```

You should see a page titled **"GitHub Pages"** with settings.

---

### 2. Look for "Build and deployment" Section

Scroll down until you see a section called **"Build and deployment"**

Under this section, you'll see:

#### **Source** (dropdown menu)
- Click the dropdown
- Select: **"Deploy from a branch"**

---

### 3. Configure Branch Settings

After selecting "Deploy from a branch", you'll see two new dropdowns appear:

#### **Branch** (first dropdown)
- Click it
- Select: **`main`** ← Your branch name

#### **Folder** (second dropdown)  
- Click it
- Select: **`/ (root)`** ← Deploy from root folder

---

### 4. Save Your Changes

- Click the **"Save"** button (appears after selecting branch)
- You should see a blue banner: **"GitHub Pages source saved."**

---

### 5. Wait for Deployment (2-3 minutes)

After saving, GitHub will:
1. Start building your site
2. Show message: **"Your site is ready to be published at..."**
3. After 1-2 minutes, change to: **"Your site is live at https://weswhite-droid.github.io/Video-AI-Project/"**

---

### 6. Verify It's Live

You'll see a green box with:
✅ **"Your site is live at https://weswhite-droid.github.io/Video-AI-Project/"**

Click the **"Visit site"** button to test!

## ❓ Troubleshooting

### "I don't see 'Build and deployment'"

**Possible reasons:**
- You're not logged into GitHub
- You don't have admin access to the repository
- You're on the wrong settings page

**Solution:**
1. Make sure you're logged in as `weswhite-droid`
2. Go directly to: https://github.com/weswhite-droid/Video-AI-Project/settings/pages
3. Check you're in the **Settings** → **Pages** tab (left sidebar)

---

### "I see 'GitHub Pages is currently disabled'"

**This is good!** It means you're in the right place.

**Follow steps 2-4 above** to enable it.

---

### "Source dropdown is missing"

If you don't see the Source dropdown:
1. Check if GitHub Pages is disabled (see above)
2. Refresh the page
3. Make sure you have pushed files to the `main` branch

---

### "Save button doesn't appear"

The Save button only appears **after** you:
1. Select a branch (main)
2. Select a folder (/ root)

Try clicking both dropdowns again.
